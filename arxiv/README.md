# arXiv submission package

Upload `arxiv-submission.zip` to arXiv as a TeX/LaTeX source submission.

The archive contains:

- `main.tex`
- `arxiv.sty`
- `references.bib`

The project compiles without external figures because all diagrams and charts are defined in TikZ/PGFPlots. arXiv has generated HTML versions automatically for TeX/LaTeX submissions since late 2023; the separate GitHub project page is provided as a stable visual companion.
